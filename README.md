# Playwright AI Learning Project

This repository contains code and exercises for learning Playwright and JavaScript basics.

## Structure

- **chapter_01_basics/**: JavaScript basics and setup verification
- **chapter_02_java_concepts/**: Java concepts (to be added)

## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/RasihaSri/playwright-ai.git
   ```
2. Navigate to the project directory:
   ```sh
   cd playwright-ai
   ```
3. Install dependencies (if any):
   ```sh
   npm install
   ```
